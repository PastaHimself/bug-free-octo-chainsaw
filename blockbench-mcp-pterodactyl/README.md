# Blockbench MCP Pterodactyl Package

This package contains Blockbench MCP integration for Pterodactyl server management.

## Contents

- **docker/**: Docker configuration for running Blockbench MCP
- **pterodactyl/**: Pterodactyl egg configuration
- **client-configs/**: MCP client configuration files for VS Code and Claude Desktop
- **.github/workflows/**: GitHub Actions workflows for CI/CD

## Setup

See the individual directories for setup instructions.
